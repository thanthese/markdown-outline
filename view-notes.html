:howto transfer a postgres database


-- create a copy of entire database


pg_dump -U postgres testgis > testgis.sql





-- alternately, create a copy of just a table


pg_dump -U postgres -t tableName testgis > testgis.sql





-- alternately, create a copy of just a table, just the data


pg_dump -U postgres -t tableName -a testgis > testgis.sql





-- uber specific example of the copy


pg_dump -U postgres -t bridges -a --column-inserts postgis > partialBridgesNov3.sql





-- load the copy


psql -U postgres -d testgis -f testgis.sql




:howto put shapefiles into postgres


The shp2pgsql command will create the raw SQL and save it to a file.


The sql will create and populate the given table.





  shp2pgsql national_shapefile_obs.shp shapetest > shapetest.sql





The psql query to push that file into the database is this (where


postgres=user, postgis=database):





  psql -U postgres -d postgis -f shapetest.sql
